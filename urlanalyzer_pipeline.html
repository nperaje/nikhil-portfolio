URLAnalyzer
Reachability Pipeline

website_url → HTTP Check → Status Classification → Excel Output

A Python automation that takes a list of SaaS product website URLs from an Excel sheet and checks whether each one is still active, redirected, or broken. It detects domain changes, identifies potential company acquisitions or rebrands by scanning page titles and Google search results, and writes colour-coded status results back inline — row by row — into the same Excel file.

SerpAPI — Acquisition & Rebrand Detection
When a domain mismatch is detected and the page title alone isn't conclusive, a targeted Google search is fired via SerpAPI. Organic results (title, URL, snippet) are scanned for acquisition keywords to confirm or rule out a company acquisition or rebrand. Capped at 100 API calls per run.
Query: "domain acquired by redirected_domain" acqui · buys · purchase is now · formerly · fka known as · join ≤ 100 API calls cap
