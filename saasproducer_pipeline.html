AI Metadata Pipeline

Website URL → Gemini AI → Metadata + Logo → Google Sheet

A Google Apps Script automation that enriches a SaaS product database using AI. For each product URL in a Google Sheet, it calls Gemini 2.5 Flash via Vertex AI to generate structured metadata — including product name, vendor, category, and type — and simultaneously fetches the product logo via Logo.dev, uploading it to a permanent CDN. Runs in batches of 50 with auto-resume across scheduled trigger executions.

⏰  Time-Based Trigger (Apps Script)
📦  Batch Size: 50 rows per run

Before calling any external API, the script checks if the metadata cell (SaaS column) and logo cell are already populated. If both exist, the row is skipped entirely — preventing duplicate API calls and unnecessary cost.

Gemini 2.5 Flash via Vertex AI
Website URL is passed to Gemini with a structured prompt. The model classifies the product and returns structured metadata parsed line-by-line.

Pipeline B — Logo Fetching
Logo.dev → Catbox CDN
Root domain is extracted from the URL, then the Logo.dev API is queried for a 500px logo. On success, the image blob is uploaded to Catbox for permanent hosting.
