Europe Fastest-Growing
Companies Pipeline

Statista API → Extract → Transform → PostgreSQL → Metabase

A Python-based ETL pipeline that extracts the FT1000 Europe's Fastest-Growing Companies 2026 ranking from Statista. By inspecting the page's network activity, the underlying JSON API endpoint was identified — allowing direct HTTP requests via Python's requests library, bypassing the need for browser automation. The raw JSON is cleaned, structured, and loaded into a local PostgreSQL database — then surfaced through a Metabase dashboard for interactive exploration of growth trends across companies, countries, and industries.

HTTP Request
Python Requests — Statista JSON API
The ranking page loads its table data via a background XHR/Fetch call, identified through browser DevTools (Network → Fetch/XHR tab). The underlying API endpoint is called directly using Python's requests library with the appropriate headers, returning a clean JSON payload — no browser automation required.
JSON API Response
② Parse
🐍
JSON Parsing
Python — Structured JSON Extraction
The JSON API response is parsed directly using Python's built-in json library. Each record in the response array is mapped into a structured Python dictionary with typed fields — rank (int), company name (str), CAGR % (float), country (str), and industry (str).
List of Dicts → DataFrame

⚙️
Data Engineering Layer
Pandas — Transformation & Data Cleaning
Raw extracted records are loaded into a Pandas DataFrame and cleaned before storage.

④ Load

Storage
PostgreSQL — Local Database
Cleaned records are written into a local PostgreSQL table using SQLAlchemy. The table schema captures all ranking fields with appropriate column types and a primary key on rank. Upsert logic prevents duplicate rows on re-runs.
