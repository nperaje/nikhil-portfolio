Product Lookup
Chrome Extension

Active Tab URL → BigQuery API → Popup Results

A Chrome Extension that reads the current tab's domain in domain.tld format, queries a BigQuery-backed REST API to search a products table for matching website_url entries, and surfaces the matched product records directly inside the extension's popup window — giving instant product intelligence without leaving the browser.

🖱️  User Clicks Extension Icon
① Extract — Active Tab URL
🌐
Browser API
Chrome Tabs API
On popup open, the extension calls chrome.tabs.query() to fetch the currently active tab. The full URL is parsed and stripped down to domain.tld format using URL parsing, ready to be passed as the query parameter.
chrome.tabs.query() URL Parsing domain.tld Extraction
② Query — BigQuery via API
🔍
API Call
REST API Request
The extracted domain is sent as a query parameter to a REST API endpoint. The API receives the domain.tld string and constructs a parameterised query to BigQuery.
GET /api?domain=domain.tld
🗄️
Data Source
BigQuery — Products Table
The API runs a SQL query against the products table in BigQuery, filtering rows where website_url contains or matches the supplied domain.tld. Returns matched product records as JSON.
JSON Response
③ Transform & Process
⚙️
Data Processing Layer
Response Parsing & State Handling
The raw JSON API response is parsed and processed inside the extension's popup script. Results are validated, empty states are handled gracefully, and data is shaped into display-ready structures.
Parse JSON Null / Empty Handling Error Handling Field Mapping Loading State Result Formatting
④ Display — Popup UI
🪟
UI Layer
Extension Popup Window
Matched product records are dynamically rendered inside the Chrome Extension popup. Results display product name, matched URL, and any additional metadata returned from BigQuery — all inline, without leaving the current page.
