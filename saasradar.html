SaaSRadar

Transaction & SSO Description → Classify → Excel Output

A Python automation that scans raw transaction and SSO descriptions to identify known SaaS products. It cleans each description, generates n-gram combinations, and matches them against a product database. Unmatched rows are further classified — flagged as ignore words, uncommon terms, or potential new SaaS entities not yet in the database — with all results colour-coded and written back to Excel.

🖥️  Tkinter UI — Manual Trigger
⚙️  run_product_mapper() — Programmatic
① Input
📂
Master File
Transaction / SSO File
User selects an Excel (.xlsx) or CSV file containing raw transaction or SSO descriptions. CSV files are auto-converted to .xlsx before processing.
.xlsx / .csv
📚
Reference Data
Products & Ignore Words DB
Two reference workbooks loaded at startup — a known SaaS products list and an ignore-words list used to flag and archive irrelevant transactions.
products.xlsx ignorewords.xlsx
② Filter & Clean
🧹
Pre-processing
Filter Unrecognised Rows & Data Cleaning
Unrecognised rows are isolated into an intermediate file. Each description is then deep-cleaned before NLP matching begins.
Strip URLs & Emails Remove Alphanumerics Strip Months & Numbers Remove Short Words (<3 chars) Remove Stop Words Normalise Whitespace
③ NLP Matching Engine
🔍
n-gram Search
Product Lookup via N-gram Combinations
For each cleaned description, 1–3 word n-gram combinations are generated and searched against the indexed products database using fast hash-map lookups.
Unigrams Bigrams Trigrams build_index() search_value_in_array()
④ Classification
✅
Match Found
SaaS Product Identified
Description matches one or more known products. Tagged as SPV (single match) or MPV / MULTIPLE_PRODUCTS.
SPV MPV MULTIPLE_PRODUCTS
🏷️
Special Flags
Ignore / Uncommon / Entity
Descriptions that don't match products are further classified — archived if they contain ignore words, flagged if words are uncommon, or probed for new entity patterns.
IGNORE_WORDS UNCOMMON_WORD ENTITY_FOUND SPEND_KEYWORD
🆕
Discovery
New SaaS Entity Detection
Regex pattern matching on remaining unclassified words extracts potential new SaaS product names not yet in the database — surfacing them for review.
find_entity() Regex Patterns Context Extraction
⑤ Output
🎨
Colour-Coded Excel
Results Written to Master File
Results are merged back into the master Excel file. Each row is colour-coded in the MAPPER_COMMENTS column for instant visual interpretation.
🟢 Green — SaaS Match 🟠 Orange — Ignore Word 🟡 Yellow — Entity Found 🔵 Cyan — Spend Keyword
🔔
Completion
Notification & Audio Alert
Once processing finishes, a desktop notification is triggered via pyautogui and an optional text-to-speech audio cue announces completion — no manual monitoring needed.
